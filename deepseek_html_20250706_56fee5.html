GitHub Pages 部署指南

通过本指南，您将学会如何注册 GitHub 账户并部署您的网页工具发布网站到 GitHub Pages

注册 GitHub

  • 访问 GitHub 注册页面
  • 创建您的用户名
  • 填写邮箱和密码
  • 完成邮箱验证

创建仓库

  • 点击 "+" 新建仓库
  • 命名规则: username.github.io
  • 设为公开仓库
  • 添加 README 文件

上传文件

  • 上传网站 HTML/CSS/JS 文件
  • 添加 index.html 作为入口
  • 可创建其他页面文件
  • 提交更改到仓库

步骤 1：注册 GitHub 账户

访问 GitHub 注册页面 并填写必要信息：

  • 用户名（将成为您网站地址的一部分）
  • 有效的电子邮箱地址
  • 安全密码

步骤 2：创建专用仓库

登录后，创建新仓库并命名为：您的用户名.github.io

例如：如果您的用户名是 webmaster，仓库名应为 webmaster.github.io

确保选择 Public（公开）并勾选 "Add a README file"。

步骤 3：上传网站文件

在仓库页面，点击 "Add file" > "Upload files"：

  • 上传您的 index.html 文件
  • 上传 CSS、JavaScript 和其他资源文件
  • 确保网站入口文件为 index.html

添加提交信息后点击 "Commit changes"。

步骤 4：启用 GitHub Pages

在仓库中：

  1. 点击顶部菜单的 "Settings"
  2. 在左侧边栏选择 "Pages"
  3. 在 "Source" 部分选择分支（通常为 main）
  4. 选择文件夹（通常为 /(root)）
  5. 点击 "Save"

步骤 5：访问您的网站

等待 1-2 分钟后，访问：

https://您的用户名.github.io

例如：https://webmaster.github.io

您的网站现在已上线！
